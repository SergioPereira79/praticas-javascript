Guia Rápido de JS

1. Manipulação do DOM (HTML)

document.getElementById('id')

Busca o único elemento com o 'id' especificado. Rápido e específico.

document.querySelector('seletor')

Busca o primeiro elemento que corresponde ao seletor CSS.

document.querySelectorAll('seletor')

Busca TODOS os elementos e retorna uma lista.

elemento.appendChild(filho)

Insere um novo elemento 'filho' dentro do 'elemento'.

2. Reagir a Ações (Eventos)

elemento.addEventListener('evento', callback)

Ouve por uma ação (evento) e executa uma função (callback).

Eventos comuns: 'click', 'scroll', 'submit', 'mouseenter', 'keydown'.

event.preventDefault()

Usado dentro do callback para impedir a ação padrão do navegador (ex: recarregar página em formulários).

3. Mudar Aparência (Classes e Estilos)

elemento.classList.add('classe')

Adiciona uma classe CSS.

elemento.classList.remove('classe')

Remove uma classe CSS.

elemento.classList.toggle('classe')

Adiciona/Remove a classe (liga/desliga).

4. Animações com JavaScript

IntersectionObserver

A melhor forma de disparar animações quando um elemento entra na tela (scroll reveal).

requestAnimationFrame(callback)

Para animações contínuas e suaves, como o efeito Parallax.

5. Lidar com Listas (Arrays)

array.forEach(item => { ... })

Executa uma ação para cada item da lista.

array.filter(item => { ... })

Cria uma nova lista apenas com os itens que passam em um teste.

array.find(item => { ... })

Encontra e retorna o primeiro item que passa em um teste.
